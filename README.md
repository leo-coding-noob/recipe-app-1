# Recipe Management System

## Description
A web-based recipe management system developed as a BCA 4th Semester Project.
The system allows users to add, edit, delete, and view recipes using CRUD operations.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Features
- Add new recipes
- Edit existing recipes
- Delete recipes
- View all recipes
- User-friendly interface

## How to Run the Project
1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder to `htdocs`
4. Import the database using phpMyAdmin
5. Open browser and visit:
   [http://localhost/recipe-app-1/](http://localhost/recipe-app/php/index.php)
